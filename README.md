# Arenamaster

Eine umfassende PvP-Erweiterung für World of Warcraft Patch 12.0.5

## Übersicht
Arenamaster ist ein leistungsstarkes Addon, das dein Arena-Erlebnis in World of Warcraft Patch 12.0.5 verbessert.

## Features
- Arena-Statistiken und Tracking
- Gegner-Informationen
- (Weitere Features folgen)

## Installation
1. Lade das Addon herunter
2. Entpacke es in deinen WoW `Interface/AddOns` Ordner
3. Aktiviere das Addon im Spiel
4. Starte World of Warcraft neu

## Anforderungen
- World of Warcraft Patch 12.0.5 oder neuer
